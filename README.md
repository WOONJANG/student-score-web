# 학생 점수 관리 웹 v6

이번 버전은 아래 두 가지를 반영했습니다.

## 1. 학생 점수 직접 반영 규칙 변경
기존:
- 하루에 전체 한 번

변경:
- **항목당 하루에 한 번**
- 예:
  - 2026-03-07에 `물마시기`를 반영했다면
  - 그날은 `물마시기`를 다시 반영할 수 없음
  - 대신 `분리수거 하기`, `화분에 물주기` 같은 다른 항목은 반영 가능

즉, 같은 날 같은 항목만 막고 다른 항목은 허용합니다.

## 2. 선생님 로그 확인 및 삭제 기능 추가
선생님 계정으로 로그인하면 학생별로
- 학생이 직접 반영한 날짜/시간
- 선생님이 직접 조정한 날짜/시간
- 반영된 항목/내용
- 반영 점수

를 순서대로 볼 수 있고,
해당 로그를 **삭제하면 점수도 함께 되돌아갑니다.**

---

## 데이터 구조
시트:
- users
- sessions
- logs

`logs` 시트에 학생 제출과 선생님 조정이 모두 남습니다.

---

## 학생 제출 방식
학생이 여러 항목을 한 번에 선택해 저장하더라도
실제 로그는 **항목별로 1줄씩** 저장됩니다.

예:
- 물마시기
- 잔반 안 남기기

를 함께 저장하면 logs에는 2줄이 기록됩니다.

이렇게 해야
- 항목별 하루 1회 제한
- 항목별 누적 합계
- 특정 항목만 삭제

가 정확하게 됩니다.

## 설치
1. 새 Google Sheets 생성
2. Apps Script에 `Code.gs` 붙여넣기
3. `SPREADSHEET_ID` 수정
4. `setupProject()` 1회 실행
5. 웹앱 배포
6. `app.js`의 `apiBaseUrl` 수정
7. GitHub Pages 업로드
